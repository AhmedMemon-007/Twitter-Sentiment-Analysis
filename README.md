# Twitter Entity Sentiment Analysis

This project performs **sentiment analysis on Twitter data with respect to named entities** using machine learning techniques. The goal is to classify the sentiment (Positive, Negative, Neutral, or Irrelevant) expressed toward specific entities mentioned in tweets.

The implementation is provided in a Jupyter Notebook.
